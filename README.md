# July 4 Coast — The App (PWA)

A full app-shell experience: bottom tab bar, sliding screens, a living
night sky on every screen, and the complete verified July 4, 2026 plan.

**SKY** — live fireworks stage (peony/ring/willow/crackle + heart shells),
tap to launch, 🎆 button = finale, hold it = heart surprise, live countdown
to 9:00 PM showtime, optional synthesized boom sound (🔇 toggle).
**COMPARE** — animated radar chart (draws itself; tap chips to isolate a
city) + shimmer scoreboard with winner stars + mood matcher.
**CITIES** — swipeable illustrated scene cards (LB skyline & Queen Mary,
HB pier & surfer, DP cliffs & sailboat — all animated) with full playbooks.
**BUDGET** — sliding segmented tiers, tappable bars, itemized cards.
**PLAN** — route-drawing map with a driving dot, burning-fuse timeline,
backup checkpoints, pack list, rules and official sources.

## Host on GitHub Pages
1. New public repo → upload ALL files in this folder to the root.
2. Settings → Pages → Deploy from branch → main / root → Save.
3. Live in ~1–2 min at https://<username>.github.io/<repo>/ — share it.
4. Install: iPhone Safari → Share → Add to Home Screen · Android → Install app.

Offline after first load (cache july4-app-v1). To update: edit index.html,
bump the cache version in sw.js, commit.
